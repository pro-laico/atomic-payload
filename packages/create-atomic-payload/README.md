# Create Atomic Payload

The `npx`/`npm create` scaffolder for new Atomic Payload projects — pick a scaffold and it copies it, installs dependencies, rebuilds sharp, and downloads fonts so you land on a working dev server. Part of [Atomic Payload](https://atomicpayload.com).

- [Documentation](https://atomicpayload.com/docs/getting-started/quick-start)
- [Source code](https://github.com/pro-laico/atomic-payload/tree/main/packages/create-atomic-payload)

## Usage

```bash
npx @pro-laico/create-atomic-payload
```

Pass `--template <name>` to pick a scaffold (`atomic-payload`, `fonts-only`, `icons-only`, `images-only`, `styles-only`); omit it for the interactive prompt. Run `--help` for all flags.
