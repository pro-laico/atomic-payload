# Atomic Payload — Icons Only Example

A minimal Atomic Payload example that demonstrates the `@pro-laico/icons` plugin in isolation — the `Icon` and `IconSet` collections plus a small Next.js page that renders them. Part of [Atomic Payload](https://atomicpayload.com).

- [Documentation](https://atomicpayload.com/docs/examples/icons-only)
- [Source code](https://github.com/pro-laico/atomic-payload/tree/main/examples/icons-only)
- [Documentation source](https://github.com/pro-laico/atomic-payload/tree/main/docs/content/docs/examples/icons-only.mdx)

## Quick start

```bash
npx @pro-laico/create-atomic-payload my-icons --template icons-only
```

## Atomic Payload packages used

- `@pro-laico/icons` — the `icon` and `iconSet` collections and the bundled `<Icon>` server component.
- `@pro-laico/core` — cache/revalidation helpers (pulled in by `@pro-laico/icons`).
